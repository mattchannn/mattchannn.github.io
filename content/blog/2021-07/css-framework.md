# CSS 框架 介紹

CSS 框架是一種軟體框架，以方便網頁開發跟確保更符合標準的開發。大部份的 CSS 框架除了提供預先寫好的 CSS 以外，還會外加一些 Javascript 的功能，以便為網頁提供互動。本文將會就幾款在 2021 年比較流行的框架進行比較。

## Bootstrap

Bootstrap 是一款由 Twitter 在 2011 年推出的開源 CSS 框架，由於定是由一系列用於前端開發的工具包組成的，因此它提供了很多內建的組件，非常易於定制(customized components)。它基於 HTML、CSS 和 JavaScript，為 Github 上最熱門的開源項目之一。

<img src="/images/blog/bootstrap-banner-resize.png" alt="Bootstrap logo" class="css-framework banner-logo"/>

- 由 Twitter 在 2011 年推出。
- 最流行的前端框架。
- 技術社區(technical community)的發展比較成熟。
- 缺點
  - 過度使用 !important。
  - 過度依靠 jQuery。

<br />
## Tailwind CSS

<img src="/images/blog/tailwindcss-banner-resize.png" alt="Tailwind css logo" class="css-framework banner-logo"/>

- 使用 atomic CSS，幾乎讓你不用做 coding。
- 此框架沒有設計框架，意味著你可以創造自己想要的元素。
- 可重用的元素
- 缺點
  - 比較難學

## Foundation

<img src="/images/blog/foundation-banner-resize.jpg" alt="Foundation logo" class="css-framework banner-logo"/>

- 風格上比較統一，較多模組化(modularized)的元素。
- 該框架實備了兩種模組，一種是支持網頁(site)，而另一種則是支持電郵(email framework)
- 缺點
  - 太難使用。
  - 相比 Foundation Framework，社群對於 Bootstrap 的支持更好．

## Bulma

<img src="/images/blog/bulma-banner-resize.png" alt="Bulma logo" class="css-framework banner-logo"/>

- UI 比較好看現代化。
- 容易使用該框架去開發網頁。
- 沒有使用 Javascript。
- 缺點
  - 缺少對可達性(accessibility)跟其他企業級的功能的支持。
